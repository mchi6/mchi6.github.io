2   PORTFOLIO DESIGN DL WIREFRAME VIEW 2013-14

New and Improved - 2014-15.   Mock ups and wireframe created in Photoshop. You can download my sample photoshop file of the wireframe by clicking on the link in the header. I try to focus on quick and easy methods to improve current sections, design new content areas, and managing overall website spacing .

Improvements include: removed Flash banner (not mobile supported), less images more CSS (load faster on mobile), removed iFrame (unreliable navigation), used mobile-first build (use of BootStrap framework), improved navigation (less clicking more viewing), and more focus on content (subheader design).
