5   TGBUILD  ART PROGRAM DOWNLOAD

Class Team Project - November 2014.   My recent project (for my 'Intro to Computational Media' course) is a variation of the silent game and a tangram drawing program. I worked in a team of four, doing UI UX design and implementation as well as some software development and testing. I also took on the role of coordinating and communicating work deadlines and meetings. Technologies used include Processing (Java) and Photoshop.

My UI UX design role -  included creating all interface design, sketches, and intial mockups.

My software development role -  included creating front-end code, timer implementation, level transition, and some object properties.
